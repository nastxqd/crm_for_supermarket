# crm_for_supermarket
Inventory Management System — веб-приложение для автоматизации учёта товаров в розничной сети. Система управляет остатками на складах, отслеживает партии и сроки годности, фиксирует перемещения товаров. Реализована ролевая модель (администратор, менеджер, кладовщик) с разграничением доступа.
