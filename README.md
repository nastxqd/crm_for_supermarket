# crm_for_supermarket
Inventory Management System — веб-приложение для автоматизации учёта товаров в розничной сети. Система управляет остатками на складах, отслеживает партии и сроки годности, фиксирует перемещения товаров. Реализована ролевая модель (администратор, менеджер, кладовщик) с разграничением доступа. 
Основной функционал:

Учет остатков товаров в реальном времени

Управление партиями и контроль сроков годности

Фиксация перемещений и списаний товаров

Ведение справочников (товары, поставщики, склады)

Разграничение доступа по ролям (админ, менеджер, кладовщик)

Технологии:

Бэкенд: Java 21, Spring Boot, Spring Security, Spring Data JPA

База данных: MySQL

Фронтенд: Thymeleaf, HTML, CSS, Bootstrap, JavaScript
